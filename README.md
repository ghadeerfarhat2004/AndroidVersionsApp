# تطبيق إصدارات الأندرويد (Android Versions App)

تطبيق أندرويد تم بناؤه باستخدام لغة جافا، ويقوم باستعراض قائمة إصدارات نظام الأندرويد مع الميزات التالية:
- استخدام الـ RecyclerView لعرض القائمة بكفاءة.
- تلوين الأسطر بشكل متبادل (أبيض ورمادي).
- التفاعل مع النقر على العناصر وإظهار رسائل Toast باسم الإصدار.
- زر مخصص لترتيب عناصر القائمة أبجدياً.
